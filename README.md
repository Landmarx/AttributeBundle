# Landmarx Geo Spatial Node Mapping System
##Attribute Bundle

Attribute bundle for the Landmarx Geo Spatial Node Mapping System.

###1. Installation

1) Composer
```js
"landmarx/attribute-bundle": "@dev",
```

```bash
$ php composer.phar update "landmarx/attribute-bundle"
```

###2. Configuration
none needed.
